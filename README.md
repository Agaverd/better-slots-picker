# MISIS Schedule Analyzer

Скрипт для анализа расписания учебных групп через API `schedule.misis.club`.

Позволяет:
- получить расписание по списку групп;
- определить, во сколько у каждой группы заканчивается последняя пара;
- собрать статистику по дням недели;
- посчитать обычную и взвешенную статистику;
- оценить, в какое время лучше проводить мероприятие для целевой аудитории.

---

## Задача

Проект нужен для выбора оптимального времени проведения мероприятий по расписанию групп.

Пример кейса:
- есть список целевых групп;
- для каждой группы нужно понять, когда у неё заканчиваются пары;
- по пятнице нужно выбрать слот, в который свободна максимальная часть аудитории;
- для разных наборов групп можно задавать разные веса.
